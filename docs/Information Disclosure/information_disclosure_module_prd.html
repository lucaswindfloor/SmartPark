1. 文档概述

1.1 项目背景与目标

作为智慧园区综合管理平台的重要组成部分，信息公开模块旨在建立一个统一的信息发布和管理平台。通过规范化的信息管理流程，提升园区信息传递效率和服务质量，实现园区管理方与企业用户之间的高效沟通。

核心目标：

  • 建立统一的信息发布渠道，实现多类型信息（通知公告、政策文件、园区活动、调查问卷、需求发布）的集中管理
  • 提供灵活的信息发布机制，支持不同公开范围和确认要求的信息发布
  • 加强园区活动管理，支持活动报名、签到、统计等全流程管理
  • 实现问卷调查功能，支持问卷设计、数据收集和统计分析
  • 建立需求发布平台，促进企业间项目合作、成果展示和人才招聘等交流

1.2 产品愿景

打造园区综合信息服务平台，实现信息发布、活动管理、调查反馈、需求对接等功能的一体化管理，促进园区信息的高效流通和资源的有效整合。

长期目标：

  • 成为园区管理方的高效工作平台
  • 成为企业用户的便捷服务窗口
  • 成为园区信息交流的核心枢纽

2. 用户研究与场景分析

2.1 园区管理方

核心需求：

  • 发布和管理各类通知公告、政策文件
  • 跟踪统计信息的查看和确认情况
  • 组织园区活动并管理报名、签到
  • 设计和发布问卷，分析调查结果
  • 审核企业发布的需求信息

典型场景：

  • 发布重要通知并要求企业确认接收
  • 发布新政策并统计企业查看情况
  • 组织园区活动并导出报名人员名单
  • 发布调查问卷并查看统计结果

2.2 企业用户

核心需求：

  • 及时接收和确认重要通知
  • 查看和下载政策文件
  • 报名参加园区活动
  • 参与问卷调查
  • 发布企业需求信息

典型场景：

  • 查看并确认接收重要通知
  • 报名参加园区活动并收到提醒
  • 发布企业项目合作需求
  • 填写园区满意度调查问卷

2.3 公众用户

核心需求：

  • 浏览园区公开信息
  • 查看园区活动信息
  • 浏览企业发布的需求

典型场景：

  • 查看园区最新动态和政策
  • 浏览园区公开活动信息
  • 查看企业招聘需求

4.3 功能详细说明

4.3.1 通知公告

功能描述: 支持通知公告管理，包含新增、删除、查询、修改、审核、发布、置顶功能；新增时，可设置标题、正文、公开范围。支持统计用户已查看数量。对于重要的通知公告，可设置接收人需要点击确认接收。

备注: 站内信推送都是已接收状态，所以已查看可以表示该用户已接收到该信息。

功能点 详细说明
通知管理
  • 新增通知：创建标题、正文、设置公开范围（全部用户/入驻企业/特定企业/仅管理员）
  • 编辑通知：修改标题、正文、公开范围、确认接收设置
  • 删除通知：批量或单条删除通知
  • 查询通知：按标题、日期、状态、公开范围等条件搜索
通知审核与发布
  • 审核通知：审核通知内容，通过或拒绝
  • 发布通知：将审核通过的通知发布供用户查看
  • 置顶设置：将重要通知置顶显示
  • 确认接收设置：通过复选框设置是否需要接收人确认接收
通知统计
  • 查看统计：显示通知查看人数、确认人数，计算确认率
  • 确认接收：重要通知要求用户点击确认按钮确认接收
  • 导出统计：将查看和确认统计数据导出Excel
  • 统计可视化：通过图表直观展示查看和确认情况

4.3.2 政策文件

功能描述: 支持政策文件管理，包含新增、删除、查询、修改、审核、发布、置顶功能；新增时，可设置标题、正文、公开范围、政策实施时间。支持统计用户已查看数量。

功能点 详细说明
政策管理
  • 新增政策：创建标题、正文、政策分类（财税政策/产业政策/人才政策）、实施时间、设置公开范围
  • 编辑政策：修改标题、正文、分类、实施时间、公开范围
  • 删除政策：批量或单条删除政策文件
  • 查询政策：按标题、日期、政策分类、实施时间等条件搜索
政策审核与发布
  • 审核政策：审核政策内容，通过或拒绝
  • 发布政策：将审核通过的政策发布供用户查看
  • 置顶设置：将重要政策置顶显示
  • 附件管理：支持上传和管理政策相关附件文档
政策统计
  • 查看统计：显示政策查看人数和下载次数
  • 政策分类：按政策类别进行统计分析
  • 导出统计：将查看统计数据导出Excel
  • 文件下载：支持政策文件和附件的下载功能

4.3.3 园区活动

功能描述: 支持园区活动管理，包含新增、删除、查询、修改、审核、发布、报名、统计、置顶功能；新增时，可设置标题、正文、公开范围、活动报名时间、活动开始时间。支持统计用户已查看、已报名数量。支持对已报名人员信息导出。

功能点 详细说明
活动管理
  • 新增活动：创建标题、正文、报名时间（开始和结束时间）、活动时间（开始和结束时间）、活动地点、设置公开范围
  • 编辑活动：修改标题、正文、时间、地点、公开范围
  • 删除活动：批量或单条删除活动
  • 查询活动：按标题、日期、状态、地点等条件搜索
活动审核与发布
  • 审核活动：审核活动内容，通过或拒绝
  • 发布活动：将审核通过的活动发布供用户查看
  • 置顶设置：将重要活动置顶显示
  • 活动状态管理：跟踪活动状态（报名中/即将开始/进行中/已结束）
活动报名管理
  • 报名管理：查看、管理活动报名情况，包括报名人员姓名、企业、联系方式等
  • 导出报名：以Excel格式导出已报名人员名单及详细信息
  • 签到管理：记录活动参与人员签到情况，支持线下扫码签到
  • 活动提醒：自动发送活动开始提醒给已报名人员
活动统计
  • 查看统计：显示活动查看人数、报名人数、出席人数
  • 报名率计算：自动计算活动报名率（报名人数/查看人数）
  • 出席率计算：自动计算活动出席率（出席人数/报名人数）
  • 统计分析：按企业、部门等维度分析活动参与情况
  • 导出统计：将活动统计数据导出Excel

4.3.4 调查问卷

功能描述: 支持用户根据权限在此进行调查问卷新增、删除、查询、修改、发布、置顶功能操作。支持用户在此进行查看参与问卷调查。支持用户在此进行查看所有调查结果。支持用户在此进行查看问卷调查各项结果统计。

功能点 详细说明
问卷管理
  • 新增问卷：创建标题、问卷说明、填写时间（开始和结束时间）、设置公开范围
  • 问卷设计：支持多种题型（单选、多选、填空、评分、矩阵等），设置必答/选答题
  • 编辑问卷：修改标题、问题、答案选项、时间范围
  • 删除问卷：批量或单条删除问卷
  • 查询问卷：按标题、日期、状态等条件搜索
  • 问卷预览：发布前预览问卷效果
问卷发布
  • 发布问卷：将问卷发布供用户填写
  • 设置截止时间：设置问卷填写截止时间
  • 置顶设置：将重要问卷置顶显示
  • 问卷状态管理：跟踪问卷状态（草稿/发布中/已结束）
  • 问卷提醒：可对未填写用户发送填写提醒
问卷回复管理
  • 查看回复：查看问卷回复详情，包括每个问题的回答
  • 回复筛选：按企业、部门、提交时间等条件筛选回复
  • 导出回复：导出问卷原始回复数据为Excel
  • 回复核验：检查问卷回复的有效性
问卷统计分析
  • 统计分析：各问题的回答统计和分析，支持平均分、分布、百分比等统计方式
  • 图表展示：用饼图、柱状图、折线图等形式可视化展示统计结果
  • 交叉分析：支持不同问题之间的交叉分析
  • 回复率统计：统计问卷发放量、回复量和回复率
  • 导出报告：将统计分析结果导出为Excel或PDF报告

4.3.5 需求发布

功能描述: 需求发布模块具备需求管理和审核发布功能。需求管理功能允许用户对企业申请发布的需求进行新增、删除、查询、修改和置顶操作，方便用户对需求进行有效管理。用户可自主发布项目合作、成果展示、招聘需求等，增强企业间的互动和合作机会。审核发布功能则由园区人员负责，对企业提交的需求进行审核，确保信息真实有效后发布，保障平台信息质量，提升用户信任度。

备注: 调研结论：原科研成果改成"需求发布"。

功能点 详细说明
需求管理
  • 新增需求：创建标题、需求描述、需求类型（项目合作/成果展示/招聘需求）、有效期、联系信息（企业名称、联系人、联系电话、电子邮箱）
  • 编辑需求：修改标题、描述、类型、联系信息、有效期
  • 删除需求：批量或单条删除需求
  • 查询需求：按标题、类型、发布企业、日期等条件高级搜索
  • 需求状态管理：跟踪需求状态（草稿/待审核/已发布/已过期）
需求审核与发布
  • 审核需求：审核需求内容的真实性和合规性，通过或拒绝
  • 发布需求：将审核通过的需求发布供用户查看
  • 置顶设置：将重要或紧急需求置顶显示
  • 审核流程：支持多级审核流程，可设置审核人和审核规则
  • 拒绝理由：拒绝审核时需填写拒绝理由，通知发布者
需求统计
  • 查看统计：显示需求查看人数、按类型统计需求数量和分布
  • 企业分布：统计发布需求的企业数量和行业分布
  • 响应情况：跟踪需求的响应和对接情况
  • 热度分析：分析最受关注的需求类型和领域
  • 导出统计：将需求统计数据导出Excel
需求对接
  • 意向表达：用户可对需求表达合作意向
  • 信息保护：对发布企业的敏感信息进行保护
  • 需求推送：根据企业特点推送相关需求
  • 对接跟踪：记录需求对接和合作进展

1. 发起申请

企业管理员王经理需要为新入职的多名员工申请停车月卡。他打开浏览器，登录公共服务平台后，进入"服务大厅" → "配套服务" → "停车月卡"功能模块。

用户行为与体验

用户行为：登录系统，导航到月卡申请页面

情感状态：专注 期待

接触点：公共服务平台 停车月卡模块

痛点：如果功能导航不清晰，可能难以找到入口

2. 填写信息

王经理选择"企业批量申请"选项，上传预先准备好的员工信息表格（包含姓名、部门、车牌号、车型等信息），系统自动解析表格并填充到申请表单中。他检查信息无误后，选择月卡类型（标准月卡）和时长（3个月）。

用户行为与体验

用户行为：上传批量信息，选择月卡类型和时长

情感状态：满意 专注

接触点：批量导入表单 月卡选择界面

痛点：如果批量导入格式有误，需要重新调整格式再上传

3. 确认费用

系统根据企业类型自动计算优惠后的费用。王经理看到系统显示的费用明细：5名员工×标准月卡×3个月=4500元，企业A类优惠95折，最终应付4275元。他仔细核对后确认费用无误。

用户行为与体验

用户行为：查看费用明细，核对优惠计算

情感状态：满意 安心

接触点：费用计算模块 优惠显示

痛点：如果企业优惠政策有变动，可能会困惑为何计算方式不同

4. 在线支付

王经理选择使用企业预付款支付方式。系统显示当前企业预付款余额充足，他点击"确认支付"按钮。系统验证通过，显示支付成功，并自动生成支付凭证和申请记录。

用户行为与体验

用户行为：选择支付方式，确认支付

情感状态：安心 满意

接触点：支付选择界面 支付结果页

痛点：如果企业预付款余额不足，需要临时切换支付方式

5. 等待审批

王经理在"我的申请"中看到刚才提交的月卡申请状态为"审批中"。系统提示预计1-2个工作日内完成审批，并会通过系统消息和短信通知审批结果。他退出系统，等待园区运营管理员的审批。

用户行为与体验

用户行为：查看申请状态，等待审批结果

情感状态：期待 耐心

接触点：申请状态页 系统通知

痛点：审批时间过长可能导致员工无法及时使用停车位

痛点：支付过程中断连可能导致状态不明确

5. 续期完成

支付成功后，系统自动更新赵先生的月卡有效期，并发送续费成功的通知。赵先生可以在"我的月卡"中查看更新后的月卡信息，包括新的到期日期和续费记录。系统会在新的到期日前再次发送到期提醒。

情感状态：满意 放心

接触点：续费成功通知 更新后的月卡信息

痛点：如果系统未及时更新权限，可能影响正常使用
